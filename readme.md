#  Dynamic Quiz Application with Timer & Result Analysis

## Overview
A responsive quiz web app built using HTML, CSS, and JavaScript that:
- Loads questions dynamically
- Includes countdown timers
- Displays detailed results and performance analysis using Chart.js

## Features
- Multiple-choice questions from an array
- Per-question countdown timer
- Result page showing correct/incorrect answers and time analysis
- Responsive design for mobile and desktop

## Automation Testing
The project includes Selenium automation scripts that:
- Launch the quiz page
- Start and navigate the quiz
- Submit automatically after last question
- Validate score and result analysis

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Testing:** Selenium WebDriver (Java)  
**Tools:** Chart.js, Eclipse/IntelliJ
